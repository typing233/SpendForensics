# SpendForensics · 个人账单侦探 🔍

一款基于 **Streamlit** 的个人账单分析小应用，帮你找出资金的每一处泄漏。

## 功能特性

| 模块 | 说明 |
|------|------|
| **自动分类** | 内置中文关键字 + 正则规则，自动将流水归入餐饮外卖、交通出行、购物电商等 11 个类别 |
| **收支总览** | 饼图 / 横向条形图展示支出构成，月度收支对比柱状图 |
| **异常检测** | 基于 Z-score 标记偏离历史均值的大额消费，散点图高亮异常点 |
| **拿铁因子** | 识别外卖、打车、咖啡奶茶、视频会员等高频小额支出，按月统计累积金额 |
| **趋势预测** | 月度支出折线图 + 线性回归趋势线 + 未来 N 个月预测区间 |
| **资金泄漏简报** | 自动生成文字简报，附储蓄率评价、预算削减建议 |

## 快速开始

### 环境要求

- Python 3.9+

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行

```bash
streamlit run app.py
```

浏览器会自动打开 `http://localhost:8501`。

## 支持的 CSV 格式

应用会自动识别以下常见列名：

| 数据字段 | 支持的列名示例 |
|----------|---------------|
| 日期 | 日期、时间、交易时间、记账日期、date |
| 描述 / 商户 | 商户名称、交易对方、备注、摘要、description |
| 金额 | 金额、交易金额、amount（也支持独立的支出 / 收入列） |
| 收支方向 | 收/支、类型、交易类型（若无此列则通过金额正负号推断） |

> 微信支付、支付宝、招商银行、工商银行等机构的标准 CSV 导出文件均可直接上传。
> 文件编码支持 UTF-8、GBK、GB18030。

## 参数调节

在左侧边栏可调整：

- **异常检测 Z-score 阈值**（默认 2.5）：值越小，检测越灵敏
- **拿铁因子最低月频次**（默认 4）：某类小额消费在单月达到该次数时触发预警
- **趋势预测月数**（默认 3）：向未来延伸的预测区间长度
